# Multi-objective-container-stacking-problem
Problem data of multi-objective container stacking problem

This repository contains the following datasets used in the MCSP study:

- The incoming containers list (organized in 50, 100, and 200 containers). Each group has 50 instances. 
- The available cranes list for each instance.
  
# Citation
If you use these data files or refer to this work, please cite:
